# horoskop
Horoscope web scraper

Scrapes the content from these Web Services:
- Krone.at
- Astroportal
- Astrowoche.de
- Horoscope.com
- Kurier.at
- Rogers.at

## Install
```
bundle install
```

## Usage:
```
ruby horoskop.rb --zodiac leo,cancer --provider krone
```

Show help
```
ruby horoskop.rb --help
```
