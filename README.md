# Horoskop
Horoscope web scraper

Scrape the content of these web services and output the result as YAML

- Krone.at
- Astroportal.de
- Astrowoche.de
- Horoscope.com
- Kurier.at
- Rogers.at

## Install
```
bundle install
```

## Usage:
```
ruby horoskop.rb --zodiac leo,cancer --provider krone
```

Show help
```
ruby horoskop.rb --help
```
